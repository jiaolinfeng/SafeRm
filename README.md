# SafeRm
Saferm
========================================================
version: 1.0
--------------------------------------------------------
brief:
srm is a safely delete tool based on rename system call.
srm command moves the files and folders to be deleted
into dustbin instead of deleting from disk directly.
--------------------------------------------------------
usage:
srm [-h] [-v]
srm file1 file2 ...
--------------------------------------------------------
install:
./install.sh
